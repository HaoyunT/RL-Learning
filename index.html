一、环境配置

我在本地环境中配置了独立的虚拟环境，用于强化学习项目开发，主要步骤如下：

  1. 添加清华镜像源： 
    conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/main
  2. 配置 Pip 镜像源： 
    pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple
  3. 创建虚拟环境： 
    conda create --prefix .\.venv python=3.11
  4. 激活虚拟环境： 
    conda activate G:\RL-Learning\.venv
  5. 安装常用包： 
    conda install numpy pandas matplotlib scikit-learn jupyterlab
  6. 安装 GPU 支持的 PyTorch： 
    conda install pytorch torchvision torchaudio pytorch-cuda=12.1 -c pytorch -c nvidia

二、GitHub 项目管理

每个强化学习项目完成后，我会上传到 GitHub 进行统一管理，步骤如下：

  1. 创建 .gitignore 文件，排除不上传的目录（如 .idea/、.venv/）。
  2. 在 GitHub 创建新仓库，并复制仓库链接。
  3. 在项目目录下执行以下命令： 
    git init
git checkout -b main
git remote add origin https://github.com/HaoyunT/RL-Learning.git
git add .
git commit -m "首次上传强化学习项目"
git push -u origin main
  4. 前往 GitHub 页面，即可看到所有代码与文件展示。
